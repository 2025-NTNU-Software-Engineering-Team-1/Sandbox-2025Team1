# HTTP 客戶端練習題

## 題目描述

請撰寫一個程式，連接到本地 HTTP 伺服器（運行在 `localhost:8080`），發送 HTTP GET 請求到 `/api/data` 端點，並輸出伺服器回應的內容。

## 輸入格式

第一行包含一個整數 `n` (1 ≤ n ≤ 10)，表示要發送的請求數量。

接下來 `n` 行，每行包含一個字串，表示要附加到請求路徑的參數（例如：`user123` 會請求 `/api/data/user123`）。

## 輸出格式

對於每個請求，輸出一行，包含伺服器回應的內容。

## 範例

### 輸入
```
2
user1
user2
```

### 輸出
```
{"id": "user1", "name": "Alice", "score": 95}
{"id": "user2", "name": "Bob", "score": 87}
```

## 注意事項

1. 伺服器運行在 `localhost:8080`
2. 請求路徑格式為 `/api/data/{參數}`
3. 伺服器會回傳 JSON 格式的資料
4. 請確保正確處理 HTTP 回應和錯誤情況

## 評分標準

- 正確連接到伺服器：20 分
- 正確發送 HTTP GET 請求：30 分
- 正確解析並輸出回應內容：50 分

## 技術要求

- **C++**: 可以使用 `libcurl` 或 `socket` 進行 HTTP 請求
- **Python**: 可以使用 `requests` 或 `urllib` 進行 HTTP 請求
