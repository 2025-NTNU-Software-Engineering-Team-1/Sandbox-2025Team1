# Trial Submission Test - Sum Two Integers

## Problem Description
Given two integers `a` and `b`, output their sum.

## Input
One line with two integers `a` and `b` (-10^9 <= a, b <= 10^9).

## Output
Print `a + b`.

## Sample Input
3 5

## Sample Output
8

## Notes
This problem is intended for trial submission pipeline testing.
