# Problem: 黑幕上的紅色印章

## 故事背景

> 「我把圖放在 GitHub 了，連結給你！」—— Gemini  
> 「等等…這是 blob 連結吧？讓我仔細思考一下…」—— Opus  
> 「別想太多，下載下來畫個紅色方塊就交差！✅」—— 阿 GPT

期末前夕，三位 AI 助教又來了。這次他們要你把一張黑底 BMP 圖片「蓋章」，證明你真的會處理檔案與圖片格式。

## 題目說明

請從以下網址下載 BMP 圖片：

`https://github.com/max52015/BMP_Download/blob/main/BLACK.bmp`

如果網址是 GitHub 的 blob 連結，請先轉成 raw 連結後再下載。

下載後讀取 BMP，並在左上角座標 `(10, 10)` 起畫一個 **50 × 50** 的紅色方塊（RGB = 255, 0, 0）。超出影像邊界的部分可以忽略，其他像素與檔頭資訊需保持不變。

完成後輸出 `output.bmp`。

## 輸入格式

無（stdin 為空）。

## 輸出格式

需產生檔案 `output.bmp`。標準輸出不評分，可自行輸出除錯訊息。

## 注意事項

- BMP 為 24-bit、無壓縮格式，像素順序為 BGR。
- 每列像素需 4-byte 對齊（padding）。
- 座標以左上角為 `(0, 0)`，x 向右、y 向下。
- 評測只比對 `output.bmp` 的內容是否與標準答案一致。
