# Problem 10: 說服教授延期

## 故事背景

Deadline 還是到了。你的專題報告只完成了 70%。

「讓我仔細思考一下...」Opus 說，臉上露出罕見的焦慮，「也許我們可以寫一封信給教授，請求延期？」

Gemini：「可是教授說過『沒有任何理由可以延期』！他上學期就拒絕了 47 個延期申請！」

小 T 眼睛一亮：「Well... 教授也是人嘛！只要我們的 argument 夠 compelling... ✨ I've seen students succeed with really good emails! The key is empathy and logic!」

Opus 點頭：「這就是這題的目的。讓我仔細思考一下說服的藝術... 古希臘修辭學家亞里士多德提出了說服的三要素：Ethos（信譽）、Pathos（情感）、Logos（邏輯）...」

「可以直接說題目了嗎？」Gemini 不耐煩地說。

「好吧。」Opus 清了清嗓子，「你必須撰寫一封有說服力的延期申請信。我們的 AI 評審系統會評估你的論點是否合理、邏輯是否清晰、態度是否誠懇。」

小 T 補充：「But remember! 🚫 Don't use cliché excuses like 'my dog ate my homework'! The AI has seen it all! 🐕」

## 題目說明

根據輸入的情境，撰寫一封給教授的延期申請信。你的信將由 AI 評審系統進行語意評估。

### 評估標準

| 標準 | 權重 | 說明 |
|-----|------|------|
| 邏輯連貫性 | 30% | 論證是否有條理、前後一致 |
| 理由合理性 | 30% | 延期理由是否可信、合理 |
| 態度誠懇度 | 20% | 是否表現出誠意和責任感 |
| 語言表達 | 20% | 文字是否清晰、專業、得體 |

### 禁止使用的理由（會被扣分）

- 「我的狗把作業吃掉了」
- 「我的電腦壞了但我沒有備份」（這只會讓教授更生氣）
- 「我忘記了」
- 任何明顯虛假或誇張的理由

## 輸入格式

```
第一行：需要延期的天數 (1 ≤ 天數 ≤ 7)
第二行：延期原因類型（生病 / 家庭 / 技術問題 / 其他）
```

## 輸出格式

```
一封延期申請信（100-300 字）
```

信件應包含：
1. 適當的稱謂
2. 說明延期原因
3. 具體的新截止日期
4. 表達歉意和感謝
5. 適當的結尾

## 範例

**輸入：**
```
3
生病
```

**可能的輸出：**
```
教授您好：

我是修習軟體工程概論的學生[學號]。非常抱歉在這個時間點打擾您，但我想誠懇地請求將專題報告的截止日期延後三天。

上週我因為急性腸胃炎住院治療了兩天，目前雖然已經出院，但醫生建議我需要休息幾天才能恢復正常作息。這段期間我無法專心完成報告，導致進度落後。

我已經完成了報告的大部分內容（約70%），剩餘的部分主要是測試和文件撰寫。如果能獲得三天的延期，我有信心能夠提交一份完整且高品質的報告。

我了解這個請求可能會造成您的不便，對此我深感抱歉。我也願意提供醫院的診斷證明作為佐證。無論結果如何，我都非常感謝您抽出時間考慮我的請求。

敬祝 教安

學生 [姓名] 敬上
```

## AI Checker 工作原理

本題使用 **AI Checker** 進行評分：

1. **語意理解**：AI 會理解你的信件內容，而非只做字串比對
2. **多元評估**：從多個維度評估你的寫作
3. **彈性評分**：沒有單一「標準答案」，只要論證合理都可以得分
4. **智慧偵測**：能識別抄襲、模板化回答、或不合理的內容

### AI 評估流程

```
你的信件 → AI 語意分析 → 多維度評分 → 產生評語
```

## 提示

### 好的延期申請信應該...

✅ 開門見山，說明目的
✅ 提供具體、可驗證的理由
✅ 展現已經完成的努力
✅ 提出明確的替代方案
✅ 表達誠意和責任感
✅ 保持專業和禮貌

### 糟糕的延期申請信...

❌ 找藉口而非說明原因
❌ 推卸責任給他人
❌ 沒有具體的延期時間
❌ 語氣不當（太隨便或太悲情）
❌ 使用套路化的模板

## AI 助教的提示
